# Configurations

We can also use React components individually to work along with Markdown syntax. Let's see how to use them:

```tsx title="ExampleButton.tsx"
import React from "react";
import styles from "./styles.module.scss";

function ExampleButton() {
  return (
    <button type="button" className={styles["example-btn"]}>
      Example Button
    </button>
  );
}

export default ExampleButton;
```

The above is absolutely written in Markdown syntax to represnt the code for `ExampleButton`, we can totally use this component `<ExampleButton />` as shown below:

<ExampleButton />
